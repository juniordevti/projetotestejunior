# Landing Page - Colégio Uperimm

Este projeto consiste no desenvolvimento de uma **Landing Page institucional para uma escola**, com o objetivo de apresentar informações sobre a instituição, sua estrutura, diferenciais e facilitar o contato com responsáveis interessados em matrículas.

O projeto foi desenvolvido como parte de um **processo seletivo para vaga de desenvolvedor**, demonstrando habilidades em HTML, CSS e JavaScript na construção de uma interface moderna, responsiva e voltada para conversão de visitantes.

---

## 📌 Funcionalidades

* Layout moderno e responsivo
* Seção hero com destaque para chamada principal
* Apresentação institucional da escola
* Seção de diferenciais educacionais
* Galeria com imagens da estrutura da escola
* Integração com **Google Maps** para localização
* Botão de contato direto via **WhatsApp**
* Animações suaves ao rolar a página
* Botões com efeitos de interação (hover)

---

## 🛠 Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **JavaScript**
* **Google Maps Embed**
* **WhatsApp API (wa.me)**
